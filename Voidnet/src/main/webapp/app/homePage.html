A WebSite To Put Space In A Box

Welcome to VoidNet


What is this place?

VoidBox.net is a site for space. I created it with the idea that it might be a portfolio I can share with others to see my work on the web. I have incorporated an array of technologies to show case including: HTML5, CSS, JavaScript, Jquery, BootStrap, and AngularJS. Not only does this website show web technology but a fullstack including: Java EE using Jax-rs to create REST services and servlets.


The inspiration for this project came from an idea that a company could have a web presence setup from top to bottom entirely free using open source material. This website is hosted on a laptop using entirely free tools like mysql for authentication and storage. Eclipse IDE for java development and to work with an edit my appserver of choice Wildfly 11. The only tool a person or company need have is a laptop and time to learn all of the parts needed to create their own website.
